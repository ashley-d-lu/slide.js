Customizable Properties



direction (*Required*)

direction determines the general direction that the set opens and closes in. It takes value 'horizontal', 'vertical', or 'diagonal'.


selectedElementPosition

selectedElementPosition determines the position of the selected element within the container. It's possible values depend on the slider's direction property. However, it's default value for all sliders is 'center'.


For horizontal sliders, selectedElementPosition takes value 'left', 'center', or 'right'

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                selectedElementPosition: 'left' or 'center' or 'right'
            });

For vertical sliders, selectedElementPosition takes value 'top', 'center', or 'bottom'

            const slider = new Slider(containerSelector, {
                direction: 'vertical',
                selectedElementPosition: 'top' or 'center' or 'bottom'
            });

For diagonal sliders, selectedElementPosition can be 'top-left', 'top-right', 'bottom-right', 'bottom-left', or 'center'. (See demo in degree section below).


degree

degree can also be customized for diagonal sliders. It determines the angle (from the horizontal) at which the set opens. Its default value is 45. It takes any value from 0 to 90. 

            const slider = new Slider(containerSelector, {
                direction: 'diagonal',
                selectedElementPosition: 'top-left',
                degree: 45 // default
            });

upOrDown

upOrDown can be customized for diagonal sliders with selectedElementPosition = 'center'. It determines whether the set opens in an upward or downward direction. It takes value 'up' or 'down'. Its default value is 'down'. 

        const slider = new Slider(containerSelector, {
            direction: 'diagonal',
            selectedElementPosition: 'center', // default
            upOrDown: 'down' // default
        });
        
        const slider = new Slider(containerSelector, {
            direction: 'diagonal',
            selectedElementPosition: 'center', // default
            upOrDown: 'up'
        });

clickToOpen

clickToOpen takes true or false. If it's value is true, the set is opened by clicking on the selected element. If it's value is false, the set is opened by hovering over the selected element. Its default value is false. 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                clickToOpen: false // default
            });
        
            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                clickToOpen: true
            });

keepOriginalOrder

keepOriginalOrder determines whether or not the elements are displayed in their original order when the set is open. It takes true or false. Its default value is false.

Try selecting different items the examples to see the effect.
            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                keepOriginalOrder: false // default

scaleFactor

scaleFactor determines how much the elements are scaled up or down in size when the set opens. It takes a number representing the scaling factor. By default, the size of the elements is not scaled when the set opens. 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                scaleFactor: 0.5
            });
        
            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                scaleFactor: 1.5
            });



animationSpeed

animationSpeed determines the speed (in milliseconds) of the animation that occurs when the set opens and closes. It takes any non-negative number. Its default value is 375. 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                animationSpeed: 600
            });
        
            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                animationSpeed: 100
            });

showSlideAnimation

showSlideAnimation determines whether the slide animation is shown. It takes true or false. Its default value is true.

showFadeAnimation

showFadeAnimation determines whether the fade animation is shown. It takes true or false. Its default value is true. 

        // Show both slide and fade animations

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            showSlideAnimation: true, // default
            showFadeAnimation: true // default
        });
        
        // Show only slide animation

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            showSlideAnimation: true, // default
            showFadeAnimation: false
        });
        
        // Show only fade animation

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            showSlideAnimation: false, 
            showFadeAnimation: true // default
        });
        
        // Show no animations

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            showSlideAnimation: false,
            showFadeAnimation: false
        });

hoverAnimation

hoverAnimation takes true or false It's default value is false. If it's value is true, when the set is open, each element performs an animation when the user hovers over it. 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                hoverAnimation: true
            });

            slider.openSetOnDOMContentLoaded();

captions

captions is used to add captions to the slider elements. captions takes a list of caption strings, which will be added to the slider elements in the same order that they initially appear in the HTML code.

Note: To be able to add captions, every element in document.querySelector(containerSelector).children must be in a non-self enclosing tag (e.g. a div). 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                captions: ['blue', 'yellow', 'red', 'green', 'orange']
            });

captionPosition

captionPosition determines the position of the captions. It's value can be 'left', 'center', or 'right'. It's default value is 'center'. 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                captions: ['blue', 'yellow', 'red', 'green', 'orange'],
                captionPosition: 'left', 'center', or 'right'
            });

onlyShowCaptionsOnHover

onlyShowCaptionsOnHover takes true or false. Its default value is false. If it's value is true, the caption for an element is only shown when the user hovers over the element. 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                captions: ['blue', 'yellow', 'red', 'green', 'orange'],
                onlyShowCaptionsOnHover: true
            });

lightbox

lightbox takes true or false. Its default value is false. Setting lightbox to true enables this library's lightbox feature. To open an element in fullscreen, the user hovers over the top right corner of the element and clicks the 'expand' icon. To exit full screen, the user hover overs the top right corner of the element and clicks the 'minimize' icon.

Note: If lightbox is true, clickToOpen is automatically set to true, since the lightbox feature works best that way.

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                lightbox: true,
                clickToOpen: true // default when lightbox = true
            });

            slider.openSetOnDOMContentLoaded();

onElementSelected

onElementSelected is a function that is called when the user selects an element. It can be assigned to any function of your choice. Its default value is a function that does nothing.

It is useful when used in conjunction with getSelectedElement (see the Functions section below) to get the currently selected element when the user changes it.

The following is a toy example where onElementSelected is assigned to a function that increments a counter variable:

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                onElementSelected: updateCounter
            });

or

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
            });
            slider.onElementSelected = updateCounter;

Functions



openSetOnDOMContentLoaded

By default, the slider is initialized to closed after the DOM loads. To initialize the slider to open, call the openSetOnDOMContentLoaded after the DOM loads. 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal'
            });

            slider.openSetOnDOMContentLoaded();

openSet

openSet opens the set.


closeSet

closeSet closes the set.


getSelectedElement

getSelectedElement returns the HTML node corresponding to the currently selected item.


Toy example using openSet, closeSet, and getSelectedElement:

  • openSet is called when the "Open" button is clicked.
  • closeSet is called when the "Close" button is clicked.
  • onElementSelected (see the Customizable Properties section above) is assigned to a function that calls getSelectedElement to get the currently selected element, logs it in the browser console, and displays its class name.
            const slider = new Slider(containerSelector, {
                direction: 'horizontal'
            });

            openButton.addEventListener('click', slider.openSet);

            closeButton.addEventListener('click', slider.closeSet);

            slider.onElementSelected = () => {
                const selectedElement = slider.getSelectedElement();
                console.log(selectedElement);
                selElemTextBox.innerText = selectedElement.className;
            };
