How To Use

Customizable Properties

direction

direction can be 'horizontal', 'vertical', or 'diagonal'.

selectedElementPosition

selectedElementPosition determines the position of the selected element within the container. It's default value is 'center'.

For horizontal sliders, selectedElementPosition can be 'left', 'center', or 'right'

        const slider = new Slider(container, {
            direction: 'horizontal',
            selectedElementPosition: 'left' or 'center' or 'right'
        });

For vertical sliders, selectedElementPosition can be 'top', 'center', or 'bottom'

        const slider = new Slider(container, {
            direction: 'vertical',
            selectedElementPosition: 'top' or 'center' or 'bottom'
        });

For diagonal sliders, selectedElementPosition can be 'top-left', 'top-right', 'bottom-right', 'bottom-left', or 'center'

degree

degree can also be customized for diagonal sliders. It determines the angle from the horizontal that the set opens in. Its default value is 45. It can take any value from 0 to 90. 

            const slider = new Slider(container, {
                direction: 'diagonal',
                selectedElementPosition: 'top-left',
                degree: 45
            });

upOrDown

urOrDown can be customized for diagonal sliders with selectedElementPosition='center'. It determines whether the set opens in an upward or downward direction. Its default value is down 

            const slider = new Slider(container, {
                direction: 'diagonal',
                selectedElementPosition: 'center', // default
                upOrDown: 'down' // default
            });
        
            const slider = new Slider(container, {
                direction: 'diagonal',
                selectedElementPosition: 'center', // default
                upOrDown: 'up'
            });

clickToOpen

If clickToOpen is true, the set is opened by clicking on the selected element. If clickToOpen is false, the set is opened by hovering over the the selected element. Its default value is false. 

        const slider = new Slider(container, {
            direction: 'horizontal',
            clickToOpen: false // default
        });
    
        const slider = new Slider(container, {
            direction: 'horizontal',
            clickToOpen: true
        });

keepOriginalOrder

keepOriginalOrder determines if the elements are displayed in their original order when the set is open. Its default value is false.

animationSpeed

animationSpeed determines the speed of the animation when the set opens and closes. Its default value is 375. 

        const slider = new Slider(container, {
            direction: 'horizontal',
            animationSpeed: 600
        });
    
        const slider = new Slider(container, {
            direction: 'horizontal',
            animationSpeed: 100
        });

showSlideAnimation

showSlideAnimation determines whether the slide animation is shown. Its default value is true.

showFadeAnimation

showSlideAnimation determines whether the fade animation is shown. Its default value is true. 

            const slider = new Slider(container, {
                direction: 'horizontal',
                showSlideAnimation: true, // default
                showFadeAnimation: true // default
            });
        
            const slider = new Slider(container, {
                direction: 'horizontal',
                showSlideAnimation: true, // default
                showFadeAnimation: false
            });
        
            const slider = new Slider(container, {
                direction: 'horizontal',
                showSlideAnimation: false, 
                showFadeAnimation: true // default
            });
        
            const slider = new Slider(container, {
                direction: 'horizontal',
                showSlideAnimation: false,
                showFadeAnimation: false
            });

onSelectedElementChanged

onSelectedElementChanged is a function that is called when the selected element changes. It's default value is a function that does nothing. You can assign it to your own function. 

        const slider = new Slider(container, {
            direction: 'horizontal',
            onSelectedElementChanged: updateCounter
        });

or

        const slider = new Slider(container, {
            direction: 'horizontal',
        });
        slider.onSelectedElementChanged = updateCounter;

Functions

openSetOnDOMContentLoaded

By default, the slider is initialized to closed when the DOM loads. Call openSetOnDOMContentLoaded when the DOM loads to initialize it to open. 

        const slider = new Slider(container, {
            direction: 'horizontal'
        });
        slider.openSetOnDOMContentLoaded();

openSet

openSet opens the set.

closeSet

closeSet closes the set.

getSelectedElement

getSelectedElement returns the HTML node corresponding to the currently selected item.

Example using openSet, closeSet, and getSelectedElement: openSet is called when the "Open" button is clicked. closeSet is called when the "Close" button is clicked. onSelectedElementChanged is assigned to a function that calls getSelectedElement, logs the selected element, and displays its class name. 

        const slider = new Slider(container, {
            direction: 'horizontal'
        });

        openButton.addEventListener('click', slider.openSet);

        closeButton.addEventListener('click', slider.closeSet);

        slider.onSelectedElementChanged = () => {
            const selectedElement = slider.getSelectedElement();
            console.log(selectedElement);
            selElemTextBox.innerText = selectedElement.className;
        }
