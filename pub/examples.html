Slide.js

How To Use

To turn a set of HTML elements into a slider:

  1. Give a unique id to each HTML element.
  2. Enclose all the HTML elements in a container div.*
  3. Set the height and width of the container div. This will determine the height and width of the slider when it is open (and how far apart the elements will be).
  4. Instantiate a Slider object, passing in the selector of the container div and an object containing customizable properties and their values. The only required property is direction, which takes value 'horizontal', 'vertical', or 'diagonal'. If a property is not specified, it will take its default value.

*Note: The library changes the position of the container div to 'relative' and the position of its child elements to 'absolute'.

Example:

HTML:

        <div id='cards' class='horizontal card-container'
            <div id='card1' class='card blue'></div>
            <div id='card2' class='card yellow'></div>
            <div id='card3' class='card red'></div>
            <div id='card4' class='card green'></div>
            <div id='card5' class='card orange'></div>
        </div>

CSS:

        .horizontal {
            height: 100px;
            width: 400px;
        }

JavaScript:

        const slider = new Slider('#cards', {
            direction: 'horizontal'
        });

Customizable Properties

direction (Required)

direction can be 'horizontal', 'vertical', or 'diagonal'.

selectedElementPosition

selectedElementPosition determines the position of the selected element within the container. Its default value is 'center'.

For horizontal sliders, selectedElementPosition can be 'left', 'center', or 'right'

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            selectedElementPosition: 'left' or 'center' or 'right'
        });

For vertical sliders, selectedElementPosition can be 'top', 'center', or 'bottom'

        const slider = new Slider(containerSelector, {
            direction: 'vertical',
            selectedElementPosition: 'top' or 'center' or 'bottom'
        });

For diagonal sliders, selectedElementPosition can be 'top-left', 'top-right', 'bottom-right', 'bottom-left', or 'center'

degree

degree can also be customized for diagonal sliders. It determines the angle (from the horizontal) at which the set opens. Its default value is 45. It takes any value from 0 to 90. 

            const slider = new Slider(containerSelector, {
                direction: 'diagonal',
                selectedElementPosition: 'top-left',
                degree: 45 // default
            });

upOrDown

urOrDown can be customized for diagonal sliders with selectedElementPosition='center'. It determines whether the set opens in an upward or downward direction. Its default value is 'down'. 

            const slider = new Slider(containerSelector, {
                direction: 'diagonal',
                selectedElementPosition: 'center', // default
                upOrDown: 'down' // default
            });
        
            const slider = new Slider(containerSelector, {
                direction: 'diagonal',
                selectedElementPosition: 'center', // default
                upOrDown: 'up'
            });

clickToOpen

If clickToOpen is true, the set is opened by clicking on the selected element. If clickToOpen is false, the set is opened by hovering over the selected element. Its default value is false. 

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            clickToOpen: false // default
        });
    
        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            clickToOpen: true
        });

keepOriginalOrder

keepOriginalOrder determines if the elements are displayed in their original order when the set is open. Its default value is false.

animationSpeed

animationSpeed determines the speed of the animation when the set opens and closes. Its default value is 375. 

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            animationSpeed: 600
        });
    
        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            animationSpeed: 100
        });

showSlideAnimation

showSlideAnimation determines whether the slide animation is shown. Its default value is true.

showFadeAnimation

showSlideAnimation determines whether the fade animation is shown. Its default value is true. 

            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                showSlideAnimation: true, // default
                showFadeAnimation: true // default
            });
        
            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                showSlideAnimation: true, // default
                showFadeAnimation: false
            });
        
            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                showSlideAnimation: false, 
                showFadeAnimation: true // default
            });
        
            const slider = new Slider(containerSelector, {
                direction: 'horizontal',
                showSlideAnimation: false,
                showFadeAnimation: false
            });

onElementSelected

onElementSelected is a function that is called when the user selects an element. Its default value is a function that does nothing. It can be assigned it to another function. 

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
            onElementSelected: updateCounter
        });

or

        const slider = new Slider(containerSelector, {
            direction: 'horizontal',
        });
        slider.onElementSelected = updateCounter;

Functions

openSetOnDOMContentLoaded

By default, the slider is initialized to closed when the DOM loads. To initialize the slider to open, call the openSetOnDOMContentLoaded after the DOM loads. 

        const slider = new Slider(containerSelector, {
            direction: 'horizontal'
        });
        slider.openSetOnDOMContentLoaded();

openSet

openSet opens the set.

closeSet

closeSet closes the set.

getSelectedElement

getSelectedElement returns the HTML node corresponding to the currently selected item.

Example using openSet, closeSet, and getSelectedElement:

  • openSet is called when the "Open" button is clicked.
  • closeSet is called when the "Close" button is clicked.
  • onElementSelected is assigned to a function that uses getSelectedElement to get the currently selected element, logs it, and displays its class name.
        const slider = new Slider(containerSelector, {
            direction: 'horizontal'
        });

        openButton.addEventListener('click', slider.openSet);

        closeButton.addEventListener('click', slider.closeSet);

        slider.onElementSelected = () => {
            const selectedElement = slider.getSelectedElement();
            console.log(selectedElement);
            selElemTextBox.innerText = selectedElement.className;
        };
