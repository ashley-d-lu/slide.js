slide.js
slide.js helps you to easily animate the opening and closing of a set of HTML elements, such as set of images or divs. It also has a lightbox feature helps you turn your set of HTML elements into a simple lightbox.

Example 1 - Photo Gallery

This example uses slide.js to create a photo gallery with a simple lightbox feature. The set opens and closes when an image is clicked. An image can also be opened in full screen by hovering over the top-right corner of an image and clicking the icon that appears.

HTML:

    <div id='lightbox' class='photos'> 

        <div id='photo1' class='photo'><
            img src='./photography/mountains.jpg'>
        </div>

        <div id='photo2' class='photo'><
            img src='./photography/bridge.jpg'>
        </div>
        
        <div id='photo3' class='photo'><
            img src='./photography/lake.jpg'>
        </div>

    </div>

Relevant CSS:

        .photos {
            height: 300px;
            width: 1230px;
        }
        
        .photo {
            height: 300px;
            width: 400px;
        }

Javascript:

        const lightboxSlider = new Slider('#lightbox', {
            direction: 'horizontal',
            clickToOpen: true,
            keepOriginalOrder: true,
            lightbox: true,
            captions: [
                'Mountains, 2020',
                'Bridge, 2020',
                'Lake, 2020'
            ],
            captionPosition: 'right',
            onlyShowCaptionsOnHover: true,
            scaleFactor: 0.8
        });
        
        lightboxSlider.openSetOnDOMContentLoaded();

Example 2 - Diagonal Photo Gallery

This example is similar to the first example, except the set opens and closes diagonally.

HTML:

    <div id='lightbox-diagonal' class='photos-diagonal'>    

        <div id='photo3' class='photo'><
            img src='./photography/mountains.jpg'>
        </div>

        <div id='photo4' class='photo'><
            img src='./photography/bridge.jpg'>
        </div>

        <div id='photo5' class='photo'><
            img src='./photography/lake.jpg'>
        </div>

    </div>

Relevant CSS:

    .photos-diagonal {
        height: 300px;
        width: 700px;
    }
    
    .photo {
        height: 300px;
        width: 400px;
    }

Javascript:

        const lightboxDiagonalSlider = new Slider('#lightbox-diagonal', {
            direction: 'diagonal',
            degree: 25,
            clickToOpen: true,
            keepOriginalOrder: false,
            lightbox: true,
            captions: [
                'Mountains, 2020',
                'Bridge, 2020',
                'Lake, 2020'
            ],
            captionPosition: 'left',
            onlyShowCaptionsOnHover: true,
            scaleFactor: 0.5
        });

        lightboxDiagonalSlider.openSetOnDOMContentLoaded();

Example 3 - Product Display

Unlike the previous examples, this example opens when selected element is hovered over instead of clicked on. When the set is open, each element also performs an animation when it is hovered over.

HTML:

    <div id='products'>

        <div id='green-iphone'>
            <img src='./productImages/iphone11-green.png' class='product'>
        </div>

        <div id='white-iphone'>
            <img src='./productImages/iphone11-white.png' class='product'>
        </div>

        <div id='black-iphone'>
            <img src='./productImages/iphone11-black.png' class='product'>
        </div>

        <div id='purple-iphone'>
            <img src='./productImages/iphone11-purple.png' class='product'>
        </div>

        <div id='yellow-iphone'>
            <img src='./productImages/iphone11-yellow.png' class='product'>
        </div>

        <div id='red-iphone'>
            <img src='./productImages/iphone11-red.png' class='product'>
        </div>

    </div>

Relevant CSS:

    #products {
        height: 100px;
        width: 600px;
    }
    
    .product {
        height: 100px;
        width: 100px;
    }

Javascript:

    const productsSlider = new Slider('#products', {
        direction: 'horizontal',
        selectedElementPosition: 'left',
        clickToOpen: false, // default
        keepOriginalOrder: true,
        captions: [
            'Green',
            "White",
            "Black",
            "Purple",
            "Yellow",
            "Red"
        ],
        captionPosition: 'center', // default
        hoverAnimation: true
    });
    
    productsSlider.openSetOnDOMContentLoaded();

This example uses slide.js to create an animated version of Twitter's sidebar menu.
