slide.js

Getting Started

  1. Download slide.js. If you are using the lightbox feature, also download /lightboxIcons and place it in the same folder as slide.js.
  2. Add 
    <script type='text/javascript' src='slide.js'></script>
<script src='https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js'></script>
    to your HTML file, and defer any scripts that will be using slide.js.

How To Use

To turn a set of HTML elements into a slider:

  1. Give a unique id to each HTML element.
  2. Enclose all the HTML elements in a container div.*
  3. Set the height and width of the container div. This will determine the height and width of the slider when it is open (and how far apart the elements will be).**
  4. Instantiate a Slider object, passing in the selector of the container div and an object containing customizable properties and their values. The only required property is direction, which takes value 'horizontal', 'vertical', or 'diagonal'. If a property is not specified, it will take its default value. (See the API to learn about the customizable properties.)

*Note: The library changes the position of the container div to 'relative' and the position of its child elements to 'absolute'.

**It is also recomended to set the height and width of the child elements of the container div, since slide.js will not automatically change their sizes to fit the container.

Example:

HTML:

            <div id='cards' class='horizontal card-container'
                <div id='card1' class='card blue'></div>
                <div id='card2' class='card yellow'></div>
                <div id='card3' class='card red'></div>
                <div id='card4' class='card green'></div>
                <div id='card5' class='card orange'></div>
            </div>

Relevant CSS:

            .horizontal {
                height: 100px;
                width: 400px;
            }

            .card {
                height: 100px;
                width: 100px;
            }

JavaScript:

            const slider = new Slider('#cards', {
                direction: 'horizontal'
            });
